Git

Git er et versjonskontrollsystem som kan brukes til å holde styr på endringer i datafiler og til å gjøre det enklere for flere folk å jobbe med de samme filene. Git fungerer på den måten at den benytter seg av flere grener som etter hvert blir en del av hovedgrenen. Dette gjør det enklere for flere folk å jobbe sammen i et prosjekt.

Github er en webbasert versjon av Git, der man kan lage repositories, og man kan velge å dele de med andre. Github gjør det enkelt for utviklere å legge ut sit arbeid og sammarbeide og kommunisere med andre utviklere.
